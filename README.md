# Audioconfig

Another small, slightly silly audio-config repo.

This repository contains a personal EasyEffects configuration for my microphone. It is not meant to be a polished, universal setup for everyone. It is simply a practical preset that I built, tweaked, and kept because it works well enough for my own use case.

The idea here is pretty simple: keep the signal chain straightforward and useful. The preset includes noise suppression, a gate to reduce background noise, an equalizer for tonal shaping, and a compressor to smooth out peaks.

Nothing here is especially fancy. There is no AI magic, no mysterious black-box processing, and no attempt to turn a simple microphone setup into a grand audio engineering project. It is mostly about learning, experimenting, and making voice audio sound a bit better without overcomplicating things.

The main file is HAVITGK59.json, which is the exported EasyEffects preset. If you want to try it, import it into EasyEffects and adjust the levels to match your microphone, room, and personal taste.

This is very much a "works for me" configuration rather than a claim that it is the perfect solution. That said, it has been good enough that I keep coming back to it.
